# Análise de Desempenho de Estruturas de Dados em Java

Este projeto tem como objetivo comparar o desempenho de diferentes estruturas de dados (Vetores, Árvores Binárias de Busca e Árvores AVL) em operações de **inserção**, **busca** e **ordenação**. Foram implementadas do zero todas as estruturas e algoritmos, sem uso de bibliotecas prontas do Java.

---

## Estrutura do Projeto
src/ │ ├── Estruturas/ │ ├── ArvoreABB.java # Implementação da árvore binária de busca (ABB) │ ├── ArvoreAVL.java # Implementação da árvore AVL (auto-balanceada) │ ├── No.java # Classe Nó utilizada pelas árvores │ └── Vetor.java # Implementação do vetor com busca sequencial, binária e métodos de ordenação (Bubble Sort e MergeSort) ├── Imagem/ │ └── imagem.png # Imagem de capa para o trabalho ├── GerarDados.java # Geração de vetores de teste (crescente, decrescente e aleatório) ├── MedidorTempo.java # Métodos para medir o tempo de inserção, busca e ordenação └── Main.java # Classe principal que executa todos os testes e imprime os resultados
